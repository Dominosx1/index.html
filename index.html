Python Basics —
Before You Start

Everything you need to know before writing a single line of code. Think of this as your orientation day.

What Even Is Python?

Python is a programming language — a way of giving instructions to a computer in a language humans can actually read. Unlike other languages that look like alien hieroglyphics, Python was designed to be readable and simple.

Real World Analogy

Think of Python like a recipe. You write down the steps (code), and the computer follows them exactly to produce a result. If you write the steps wrong, you get an error — just like a bad recipe gives you a bad meal.

Setting Up Your Environment

1

Download Python

Go to python.org and download Python 3. Always Python 3, never Python 2.

2

Download VS Code

Go to code.visualstudio.com. This is where you'll write your code — think of it as Microsoft Word but for code.

3

Install the Python Extension

Open VS Code, click the Extensions icon on the left sidebar, search "Python" and install the one by Microsoft.

4

Create Your First File

Create a file called hello.py, type print("Hello World"), then run it.

Variables — Storing Information

A variable is like a labelled box where you store information. You give the box a name, put something in it, and refer to it later by that name.

variables.py 
# Creating variables
name = "Alice"         # text (string)
age = 21              # whole number (integer)
gpa = 3.7             # decimal (float)
is_student = True    # yes/no (boolean)

print(name)           # prints: Alice
print(age + 5)       # prints: 26

Print & Input

print_input.py 
# print() — show something on screen
print("Hello World")
print(f"My name is {name} and I am {age} years old")

# input() — get something from user
name = input("What is your name? ")
age = int(input("How old are you? "))   # wrap in int() for numbers
mark = float(input("Enter mark: "))      # float() for decimals
Common Mistake

input() always returns TEXT even if the user types a number. Always use int() or float() when you need to do maths with it!

What It Does

The user enters their name and how many courses they have. For each course they enter the name and mark. The program adds up all marks, divides by the number of courses to get the average, then assigns a grade (A/B/C/D/F) and displays a clean formatted report.

The Big Idea
Think of it like this

Imagine your lecturer's gradebook. Each student has multiple subject marks. The gradebook adds them all up, divides to get an average, and prints out a report card. That's exactly what this program does — automatically.

Step by Step
  • 1
    Get student name — use input() to get the name and store it in a variable
  • 2
    Loop through courses — use a for loop with range(num_courses) to ask for each course name and mark
  • 3
    Store in dictionary — courses[course_name] = mark so each course maps to its mark
  • 4
    Calculate — total = sum(courses.values()) and average = total / len(courses)
  • 5
    Assign grade — if/elif/else checks where the average falls
  • 6
    Display neatly — use print("="*35) to make separator lines and f-strings to show values

input() + int() / float() for loop + range() dictionary sum() and len() if / elif / else f-strings and :.2f

import string
import random

length = int(input('Enter password length: '))
use_upper   = input('Include uppercase letters? (yes/no): ').lower()
use_numbers = input('Include numbers? (yes/no): ').lower()
use_symbols = input('Include symbols? (yes/no): ').lower()

characters = string.ascii_lowercase
if use_upper   == 'yes': characters += string.ascii_uppercase
if use_numbers == 'yes': characters += string.digits
if use_symbols == 'yes': characters += string.punctuation

password = ''
for i in range(length):
    password += random.choice(characters)

print('='*35)

print(f' Password : {password}')
print(f' Length   : {length}')
print(f" Uppercase: {'Yes' if use_upper == 'yes' else 'No'}")
print(f" Numbers  : {'Yes' if use_numbers == 'yes' else 'No'}")
print(f" Symbols  : {'Yes' if use_symbols == 'yes' else 'No'}")

library_management.py 
library = {}

def add_book():
    title  = input('Enter book title: ')
    author = input('Enter author name: ')
    if title in library:
        print(f"'{title}' already exists in the library!")
    else:
        library[title] = {'author': author, 'available': True}
        print(f"'{title}' by {author} added successfully!")

def borrow_book():
    view_books()
    title = input('Enter book title to borrow: ')
    if title not in library:
        print(f"'{title}' not found in the library!")
    elif library[title]['available'] == False:
        print(f"'{title}' is currently not available!")
    else:
        library[title]['available'] = False
        print(f"You have borrowed '{title}'!")

def return_book():
    view_books()
    title = input('Enter book title to return: ')
    if title not in library:
        print(f"'{title}' not found in the library!")
    elif library[title]['available'] == True:
        print(f"'{title}' was not borrowed!")
    else:
        library[title]['available'] = True
        print(f"'{title}' has been returned successfully!")

def view_books():
    print('='*45)
    print(' LIBRARY BOOKS')
    print('='*45)
    if not library:
        print('No books in the library yet!')
    else:
        for title, info in library.items():
            status = 'Available' if info['available'] else 'Borrowed'
            print(f'Title  : {title}')
            print(f"Author : {info['author']}")
            print(f'Status : {status}')
            print('-'*45)
    print('='*45)

while True:
    print('1. Add New Book  2. Borrow a Book')
    print('3. Return a Book 4. View All Books  5. Exit')
    choice = input('Enter your choice: ')
    if choice == '1': add_book()
    elif choice == '2': borrow_book()
    elif choice == '3': return_book()
    elif choice == '4': view_books()
    elif choice == '5': print('Goodbye!'); break
    else: print('Invalid choice!')

import requests

city = input('Enter city name: ')

geo_url = f'https://geocoding-api.open-meteo.com/v1/search?name={city}&count=1'
geo_response = requests.get(geo_url)
geo_data = geo_response.json()

if 'results' not in geo_data:
    print(f"City '{city}' not found! Please check the name.")
else:
    lat     = geo_data['results'][0]['latitude']
    lon     = geo_data['results'][0]['longitude']
    country = geo_data['results'][0]['country']

    weather_url = (
        f'https://api.open-meteo.com/v1/forecast?latitude={lat}'
        f'&longitude={lon}&current_weather=true'
        f'&hourly=relativehumidity_2m,apparent_temperature'
    )
    weather_data = requests.get(weather_url).json()

    temp       = weather_data['current_weather']['temperature']
    wind_speed = weather_data['current_weather']['windspeed']
    humidity   = weather_data['hourly']['relativehumidity_2m'][0]
    feels_like = weather_data['hourly']['apparent_temperature'][0]

    print('='*45)
    print(' WEATHER REPORT')
    print('='*45)
    print(f' City        : {city}, {country}')
    print(f' Temperature : {temp}C')
    print(f' Feels Like  : {feels_like}C')
    print(f' Humidity    : {humidity}%')
    print(f' Wind Speed  : {wind_speed} km/h')
    print('='*45)

calculator.py 
import tkinter as tk

window = tk.Tk()
window.title('Simple Calculator')
window.resizable(False, False)

display = tk.Entry(window, width=20, font=('Arial', 24),
                   justify='right', bd=10)
display.grid(row=0, column=0, columnspan=4, padx=10, pady=10)

def click(button):
    if button == '=':
        try:
            result = eval(display.get())
            display.delete(0, tk.END)
            display.insert(tk.END, str(result))
        except:
            display.delete(0, tk.END)
            display.insert(tk.END, 'Error')
    else:
        display.insert(tk.END, button)

def clear():
    display.delete(0, tk.END)

def delete():
    current = display.get()
    display.delete(0, tk.END)
    display.insert(tk.END, current[:-1])

buttons = ['7','8','9','/',
           '4','5','6','*',
           '1','2','3','-',
           '0','.','=','+']

row, col = 1, 0
for button in buttons:
    tk.Button(window, text=button, width=5, height=2,
              font=('Arial', 18),
              command=lambda b=button: click(b)
              ).grid(row=row, column=col, padx=5, pady=5)
    col += 1
    if col > 3:
        col = 0
        row += 1

tk.Button(window, text='C', width=5, height=2, font=('Arial', 18),
          bg='red', fg='white', command=clear).grid(row=row, column=0, padx=5, pady=5)
tk.Button(window, text='DEL', width=5, height=2, font=('Arial', 18),
          bg='orange', command=delete).grid(row=row, column=2, padx=5, pady=5)

window.mainloop()

expense_tracker.py 
expenses   = []
categories = ['Food', 'Transport', 'Shopping', 'Entertainment', 'Health', 'Other']

def add_expense():
    print('\nCategories:')
    for i, category in enumerate(categories, 1):
        print(f'  {i}. {category}')
    try:
        cat_choice = int(input('Choose category (1-6): '))
        if 1 <= cat_choice <= len(categories):
            category    = categories[cat_choice - 1]
            description = input('Enter description: ')
            amount      = float(input('Enter amount: '))
            expenses.append({'description': description,
                             'amount': amount,
                             'category': category})
            print('Expense added!')
    except ValueError:
        print('Invalid input!')

def view_expenses():
    if not expenses:
        print('No expenses recorded yet!')
    else:
        for i, expense in enumerate(expenses, 1):
            print(f"{i}. {expense['description']}")
            print(f"   Category : {expense['category']}")
            print(f"   Amount   : ${expense['amount']:.2f}")

def view_total():
    print('='*50)
    print('          SPENDING SUMMARY')
    print('='*50)
    for category in categories:
        category_total = sum(e['amount'] for e in expenses if e['category'] == category)
        if category_total > 0:
            print(f'  {category:<15}: ${category_total:.2f}')
    total = sum(e['amount'] for e in expenses)
    print('-'*50)
    print(f"  {'TOTAL':<15}: ${total:.2f}")
    print('='*50)

def show_menu():
    print('\n1. Add Expense  2. View Expenses  3. View Total  4. Exit')

while True:
    show_menu()

Chapter 04

Common Errors &
How to Fix Them

Errors are normal. Every programmer sees them every day. Here's what the most common ones mean.

Always Read Bottom to Top

When Python crashes, read the error from bottom to top. The last line is the actual error. Lines above show where it happened.

SyntaxError — Missing colon or bracket

fix it 
# WRONG
if age > 18    # missing colon!
# CORRECT
if age > 18:   # always put colon after if, for, while, def

NameError — Variable doesn't exist or is misspelled

fix it 
# WRONG — using before creating
print(Name)     # Name vs name — Python is case sensitive!
# CORRECT
name = "Alice"
print(name)

TypeError — Mixing text and numbers

fix it 
# WRONG
age = input("Age: ")     # age is TEXT "21"
print(age + 5)          # ERROR! "21" + 5 doesn't work
# CORRECT
age = int(input("Age: "))  # convert to number first
print(age + 5)            # works! prints 26

IndentationError — Wrong spacing

fix it 
# WRONG
if age > 18:
print("Adult")    # not indented!
# CORRECT — always 4 spaces inside blocks
if age > 18:
    print("Adult")
Pro Tip

When you get an error you don't understand, copy the last line and paste it into Google. Someone has definitely had the same problem. Stack Overflow at stackoverflow.com is your best friend.

Quick Help

Troubleshooting
FAQ

My code won't run — now what? Check here first before panicking.

My code won't run at all

Check This

Is the file saved as .py? Did you run the right filename? Try python filename.py exactly as named. Make sure you're in the correct folder in terminal.

SyntaxError

What It Means

Python found something it can't understand — usually a missing colon : after if, for, def, or an unclosed bracket/quote. Check the line Python mentions and the line just above it.

IndentationError

What It Means

The spacing is wrong inside a block. Everything inside an if, for, or def needs exactly 4 spaces of indent. Never mix tabs and spaces.

NameError: name 'x' is not defined

What It Means

You're using a variable before creating it, or you spelled it wrong. Python is case-sensitive — Name and name are different things.

TypeError: can only concatenate str (not "int")

What It Means

You're trying to mix text and numbers. Wrap numbers with str() like print("Age: " + str(age)), or better — use an f-string: print(f"Age: {age}").

ValueError: invalid literal for int()

What It Means

You typed something that can't be converted to a number — like typing "twenty" when the program asks for a number. Always wrap int(input(...)) in a try/except block.

IndexError: list index out of range

What It Means

You're trying to access a list item that doesn't exist. If your list has 3 items, valid indexes are 0, 1, 2. Trying list[3] or beyond will crash.

KeyError

What It Means

You're looking for a key in a dictionary that doesn't exist. Check your spelling — dictionary keys are case-sensitive. Use .get("key") to safely check without crashing.

Golden Rule

Always read the error message from the bottom up. The last line tells you what went wrong. The line above it tells you exactly where. Google the last line — someone has fixed it before you.
